- heading "Introduction" [level=2]
- text: A JavaScript UI Library
- paragraph: Bitwrench is a JavaScript library that represents HTML and CSS as plain JavaScript objects. Instead of writing HTML strings, you create UI using data structures called TACO (for HTML) and SRMC (for CSS). This approach requires no compilers, no JSX, no TSX, and no Babel - just pure JavaScript that runs directly in the browser.
- paragraph: The library provides a straightforward way to create dynamic user interfaces by treating UI elements as data that can be easily manipulated and updated.
- heading "Key Features" [level=3]
- list:

  - listitem: Direct DOM manipulation for efficient updates
  - listitem: Component system with lifecycle hooks
  - listitem: Works in modern browsers without transpilation
