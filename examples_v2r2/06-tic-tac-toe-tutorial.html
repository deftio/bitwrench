TACO Format

Bitwrench describes HTML using plain JavaScript objects called TACOs:

{ t: 'div', a: { class: 'card' }, c: 'Hello' }
//  Tag       Attributes            Content

The Render Pattern

  1. State - plain JS object holding your data
  2. Render - function that builds TACOs from state
  3. Event - handler that mutates state, then calls render
  4. Mount - bw.DOM(selector, taco) puts it on the page

No virtual DOM, no JSX, no build step. Just JavaScript.
