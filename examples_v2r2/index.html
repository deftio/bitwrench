Welcome to Bitwrench

Welcome! Bitwrench is a JavaScript library for building user interfaces using plain JavaScript objects. Instead of HTML strings or JSX, you create UI components using TACO (Tag-Attributes-Content-Options) objects and style them with SRMC (Selector-Rules-Media-Children) objects. No build tools, no transpilers - just JavaScript that runs directly in your browser.

Getting Started

  1. Include Bitwrench - Add bitwrench.js to your HTML page via script tag
  2. Learn the Basics - Start with example 0 below to understand TACO and SRMC
  3. Explore Components - Work through the examples in order to learn component patterns
  4. Build Something - Use the patterns you've learned to create your own UI

How to Use These Examples

Each example page includes live demos with tabbed views showing both the result and the code. You can copy any code snippet and experiment with it. The examples progress from basic concepts to advanced patterns, so it's best to start with the fundamentals and work your way through.
