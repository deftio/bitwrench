What You'll Learn

In this tutorial you will build a fully working tic tac toe game from scratch using bitwrench. You will start with a static 3×3 board, add click handling and turn-based state, wire up win detection and draw detection, bolt on time-travel (undo any move), and finish with a decoupled scoreboard powered by pub/sub messaging. Each step adds exactly one concept so you can see how the pieces fit together.

Here is what each step covers:

3Game logicPure functions, conditional rendering

TACO Format

Bitwrench describes HTML using plain JavaScript objects called TACOs:

{ t: 'div', a: { class: 'card' }, c: 'Hello' }
//  Tag       Attributes            Content

The Render Pattern

  1. o.state — declare initial state inside the TACO
  2. o.render — function that builds UI from el._bw_state
  3. Event handler — mutate state, then call bw.update(el)
  4. bw.DOM() — mount the TACO to the page

No virtual DOM, no JSX, no build step. Just JavaScript.
