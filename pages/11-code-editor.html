Code Editor

Lightweight syntax highlighting and editable code blocks

Overview

bitwrench-code-edit is an optional addon that provides syntax highlighting for JavaScript, CSS, and HTML. It ships as a separate file (~11 KB min) and attaches bw.highlight() and bw.codeEditor() to the bitwrench namespace.

Scope

This is a lightweight utility for rendering readable code snippets in bitwrench apps. It is not a replacement for full-featured libraries like Prism.js or highlight.js. If you need broad language support, line numbers, or advanced editor features, use one of those instead. bitwrench-code-edit covers the common case: JS, CSS, and HTML with zero dependencies.

Installation

Browser (script tag)

Load after bitwrench.umd.js. It auto-attaches to window.bw.

ES Module

Available Formats

bw.highlight(code, lang)

Tokenizes a code string and returns an array of TACO objects with syntax-colored <span> elements. Supported languages: 'js', 'css', 'html'.

Creates a contenteditable code block with live syntax highlighting. Returns a TACO object that can be mounted with bw.DOM().
