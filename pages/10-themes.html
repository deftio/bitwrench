Themes

Generate complete, scoped themes from seed colors with bw.generateTheme().

Theme Generator

Pick three seed colors, choose spacing and border-radius presets, then click Generate. The preview below updates live with fully themed buttons, alerts, badges, cards, forms, and a table.

Pre-built Theme Gallery

Click any preset to apply it to the preview area above. Each theme is generated from just three seed colors using bw.generateTheme().

Multi-theme Demo

Multiple themes can coexist on the same page. Each container gets a CSS scope class and renders the same components with different palettes.

Code Export

The JavaScript call and generated CSS for your current theme. Copy the CSS to use in production without any runtime cost.

Dark Mode + Theme Interaction

bw.toggleDarkMode() derives dark colors from your active palette. Generate a custom theme, then toggle dark mode — the dark variant uses your theme's hues, not generic grays.
