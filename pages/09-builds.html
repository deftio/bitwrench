Installation

npm

npm install bitwrench

Browser (script tag)

<script src="dist/bitwrench.umd.min.js"></script>
<link rel="stylesheet" href="dist/bitwrench.css">

ES Module import

import bw from 'bitwrench';

CommonJS require

const bw = require('bitwrench');

GitHub

git clone https://github.com/deftio/bitwrench.git

ES5 (legacy browsers)

<script src="dist/bitwrench.es5.min.js"></script>

CDN

Use bitwrench directly from a CDN — no install, no build step. Just add a script tag.

unpkg

<script src="https://unpkg.com/bitwrench@2/dist/bitwrench.umd.min.js"></script>
<link rel="stylesheet" href="https://unpkg.com/bitwrench@2/dist/bitwrench.css">

jsdelivr

<script src="https://cdn.jsdelivr.net/npm/bitwrench@2/dist/bitwrench.umd.min.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bitwrench@2/dist/bitwrench.css">

ES Module via CDN

import bw from 'https://unpkg.com/bitwrench@2/dist/bitwrench.esm.min.js';

Version pinning

@2 — latest 2.x release (recommended)
@2.0.3 — exact version
@latest — latest release (may include breaking changes across major versions)
