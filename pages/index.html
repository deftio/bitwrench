Build UI from plain JavaScript objects

A lightweight library for generating HTML from native JS objects. No JSX, no virtual DOM, no build step. Ships under 25 KB gzipped with zero dependencies.

Get Started GitHub
npm install bitwrench

How It Works

Describe your UI as plain JavaScript objects using the TACO format — Tag, Attributes, Content, Options. Nest them, loop over them, compose them. Then render to an HTML string with bw.html() or mount directly to the DOM with bw.DOM(). 

const card = {
  t: 'div', a: { class: 'bw-card' },
  c: [
    { t: 'h3', a: { class: 'bw-card-title' },
      c: 'Hello bitwrench' },
    { t: 'p', c: 'UI as native JS objects.' }
  ]
};

bw.DOM('#app', card);

What You Get

30+ Ready-Made Components

Cards, buttons, tables, forms, alerts, badges, navbars, spinners, progress bars, and more. Each one is a single function call that returns a TACO object you can customize or compose.

Palette-Driven Theming

Pick 2–3 seed colors and bw.generateTheme() derives a complete palette: buttons, alerts, badges, hover states, focus rings, and dark mode. Choose from 12 built-in presets or create your own.

Server + Client Rendering

bw.html() renders TACO objects to HTML strings for SSR, emails, or static pages. bw.DOM() mounts them as live DOM. Same objects work on both sides.

State Management Built In

Use bw.patch() for targeted DOM updates, bw.update() for re-renders, and bw.pub()/bw.sub() for decoupled event communication between components.

CLI for Markdown & HTML

Convert Markdown or HTML files to styled, self-contained pages with one command. Apply themes, inject bitwrench for interactivity, or generate plain static output.

Works Everywhere

Under 25 KB gzipped. Zero runtime dependencies. Runs in modern browsers, IE11, Node.js, and embedded environments. Include one script tag and start building.
